Meus Shows 🎬

Crie e gerencie a estrutura de suas sessões cinematográficas.

💡 Um "Show" é a estrutura completa de uma sessão: blacks, trailers, propaganda e filme.

Como Criar um Show do Zero

Passo a Passo Profissional

  1. Definir Nome do Show: "SALA 01 - DUNE 2 - 20:30"
  2. Criar BLACK Inicial (30s): Prepara a sala antes da sessão
  3. Adicionar Trailers: Em ordem, sempre em Flat 2D
  4. Adicionar Propagandas: Anúncios da sessão
  5. Criar BLACK de Transição (5s): Prepara para o filme
  6. Adicionar Filme Principal: Verificar formato (Scope/Flat)
  7. Criar BLACK Final (30s): Encerra a sessão

Programe horários e salas para suas sessões.

Como Criar um Schedule

Passo a Passo

  1. Selecione o Show: Escolha um show já criado
  2. Selecione a Sala: Em qual sala vai rodar
  3. Defina o Horário: Que horas a sessão começa
  4. Defina a Data: Qual dia vai rodar
  5. Sincronize com TMS: Envie para o Theatre Management System
  6. Confirme: Sessão pronta para rodar

Ingest 📥

Copie filmes para o servidor. Ingest é o pilar de tudo!

⚠️ SEM INGEST, SEM FILME. SEM FILME, SEM SESSÃO. Ingest é a base de toda a operação!

Como Fazer Ingest Profissional

Passo a Passo Completo

  1. Prepare o Arquivo: DCP em HD externo ou via rede TMS
  2. Conecte o Servidor: GDC, Dolby ou Christie
  3. Inicie o Ingest: Clique em "Iniciar Ingest" no painel
  4. Monitore o Progresso: Acompanhe em tempo real
  5. Valide Integridade: Servidor verifica se tudo está OK
  6. Confirme Sucesso: Se tudo OK, filme está pronto para usar
  7. Se Erro: Verifique o log e tente novamente ou escale

KDM Manager 🔐

Gerencie chaves de segurança dos filmes.

💡 KDM é a "chave de segurança" do filme. Define QUAL servidor, EM QUAL SALA e POR QUANTO TEMPO o filme pode rodar.

O que é KDM?

KDM = Key Delivery Message

  • É um arquivo de segurança que libera o filme
  • Controla qual servidor pode abrir o filme
  • Define em qual sala o filme pode rodar
  • Tem data de validade (começo e fim)
  • Se vencido ou errado: FILME NÃO ABRE

Como Importar e Gerenciar KDM

Passo a Passo

  1. Receba o KDM: Da distribuidora do filme
  2. Verifique a Data: Validade começa e termina quando?
  3. Verifique o Servidor: É para qual servidor? (GDC, Dolby, Christie)
  4. Importe no Sistema: Clique em "Importar KDM"
  5. Valide: Sistema confirma se está OK
  6. Libere o Filme: Agora o filme pode rodar

Troubleshooting 🔧

Resolva problemas comuns de operação.

Problemas Comuns e Soluções

❌ Problema: Filme Não Abre

Possíveis Causas:

  • KDM vencido ou inválido
  • KDM para servidor errado
  • Relógio do servidor desincronizado
  • Arquivo DCP corrompido

Solução:

  1. Verifique data e hora do servidor
  2. Confirme se KDM é para este servidor
  3. Verifique validade do KDM
  4. Se erro persistir, refaça o ingest

❌ Problema: Tela Preta

Possíveis Causas:

  • Projetor desligado
  • Cabo HDMI desconectado
  • Macro de automação não executou
  • Lâmpada queimada

Solução:

  1. Verifique se projetor está ligado
  2. Confirme conexão dos cabos
  3. Teste manualmente a lâmpada
  4. Execute macro manualmente

❌ Problema: Sem Som

Possíveis Causas:

  • Automação não ativou áudio
  • Processador de som desligado
  • Canal errado (5.1 vs 7.1)
  • Volume zerado

Solução:

  1. Verifique volume no servidor
  2. Confirme processador de som ligado
  3. Teste canal de áudio correto
  4. Execute automação manualmente

❌ Problema: Filme Travando

Possíveis Causas:

  • Erro no arquivo DCP
  • HD do servidor com problema
  • Ingest incompleto
  • Arquivo corrompido

Solução:

  1. Verifique integridade do DCP
  2. Teste outro filme para confirmar
  3. Se problema persistir, refaça ingest
  4. Escale para técnico de hardware
