📋 OPÇÃO 1: INSTALAÇÃO COMPLETA (COM MONGODB)

1. Abra o terminal na pasta do projeto

2. Instale as dependências:
   npm install

3. Crie o arquivo .env:
   cp .env.example .env

4. Certifique-se que o MongoDB está rodando:
   mongod

5. Inicie o servidor:
   npm run dev

6. Acesse no navegador:
   http://localhost:3000

═══════════════════════════════════════════════════════════════

🚀 OPÇÃO 2: INÍCIO RÁPIDO (SEM MONGODB)

A plataforma funciona MESMO SEM MongoDB instalado!

1. Instale as dependências:
   npm install

2. Inicie:
   npm run dev

3. Acesse:
   http://localhost:3000

(Os dados ficarão em memória temporariamente)

🎓 COMO USAR:

1. PRIMEIRO ACESSO:
   - Clique em "Criar Conta"
   - Preencha: Nome, Email, Senha (mín. 6 caracteres)
   - Aceite os termos
   - Clique em "Criar Conta"
   - Você será redirecionado ao Dashboard ✅

2. ACESSAR AS AULAS:
   - No Dashboard, clique em "Ver Aulas"
   - OU acesse: /pages/aula.html

3. VER OS ALERTAS DO SISTEMA:
   - No Dashboard, role para baixo
   - Veja o widget de Storage (81.7%)
   - Alertas aparecem automaticamente

🚨 SOLUCIONANDO PROBLEMAS:

PROBLEMA: "Porta 3000 já em uso"
SOLUÇÃO: Edite .env e mude PORT=3001

PROBLEMA: "MongoDB connection error"
SOLUÇÃO: Não tem problema! A plataforma funciona sem MongoDB

PROBLEMA: "Token inválido" ao fazer login
SOLUÇÃO: Limpe o cache do navegador (F12 > Console > localStorage.clear())
